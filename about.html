About SortScape

Empowering algorithms through visibility.

The Project

SortScape was born from a desire to make learning computer science fundamentals a beautiful, engaging, and highly interactive experience. Typical algorithmic studying often consists of dry textbooks and abstract concepts. With SortScape, the goal is to map these abstract logical processes directly into your visual cortex.

SortScape runs purely in the browser utilizing modern Javascript DOM-manipulation alongside asynchronous execution delays to properly demonstrate exactly when data compares, swaps, or locks in place.

Features

  • 8 implemented algorithms: from Bubble sort to Cocktail sort.
  • Dynamic array size scaling.
  • Variable visualization run speed.
  • Customized algorithmic data reporting: comparisons and swaps track exactly what the script is doing.
  • An interactive Manual Sorting Game to test your intuition.
  • Fully responsive, mobile-friendly design with Dark/Light mode capabilities.
