Welcome to SortScape

SortScape is an interactive playground for visualizing how different sorting algorithms operate on an array of numbers under the hood.

  • Generate Array: Creates a randomized dataset. Adjust the Array Size slider to change the number of elements.
  • Sorting Speed: Controls the animation delay. Slide to the right for faster visualizations!
  • Stop: Gracefully halts the execution of an ongoing sorting algorithm, freezing the array in its current state.
  • Manual Mode: Become the algorithm yourself! Toggle manual mode and drag & drop the array bars into their sorted order. If you accomplish the feat, you win!
